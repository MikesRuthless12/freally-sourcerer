Freally 0.20.1 — The auto-updater, now actually working

Signed auto-updater + per-OS installers. Instant cross-platform search across four lenses — filename, full-text content, audio attributes, and fuzzy name — fed by a realtime filesystem journal. The next Everything, for Windows, macOS, and Linux.

0.20.1 is a bugfix release, and everyone on an older build should take it. The updater pointed at a repository that does not exist, so no install has ever been able to find an update — and because that address is compiled into the binary, this is the first build that can. It also upgrades quick-xml, closing two advisories reachable from a hostile Office document, and repairs every Help-menu link.

What's included

  • Filename lens — instant filename, glob, and regex search on multi-million-file disks with sub-16 ms P99 results.
  • Content lens — full-text search across PDFs, Office documents, source code in 32 languages, archives (peeked without extracting), and structured files like JSON, CSV, and YAML.
  • Audio lens — search audio files by duration, LUFS loudness, codec, sample rate, silence ratio, and dynamic range using query modifiers like lufs:<-14 and codec:flac.
  • Similarity lens — find near-duplicate filenames via bigram MinHash. "I named it something like…" actually works.
  • Full Settings dialog modeled on voidtools' Everything Options — 26 panels, left-tree navigation, per-OS volume detection.
  • 18 languages, live-switching, with automatic right-to-left layout for Arabic.
  • Zero telemetry, no accounts, no cloud. Your index never leaves your machine.
